# Tutoriel : Gestion des Utilisateurs

Cette page permet aux administrateurs de gérer les utilisateurs du système.

## Étape 1 : Visualisation des Utilisateurs

- Tous les utilisateurs sont affichés dans un tableau.
- Vous pouvez filtrer, trier et rechercher les utilisateurs grâce aux contrôles en haut.

## Étape 2 : Modification des Informations

- Cliquez sur un utilisateur pour ouvrir la fiche détaillée.
- Modifiez le nom, l'email, le rôle, le statut.
- Enregistrez les modifications avec le bouton "Enregistrer".

## Étape 3 : Attribution des Rôles

- Vous pouvez attribuer des rôles comme "Admin", "Jury", "Producteur".
- Les rôles déterminent les permissions et les fonctionnalités disponibles.

## Étape 4 : Gestion du Statut

- Vous pouvez activer ou désactiver des utilisateurs.
- Les utilisateurs désactivés ne peuvent pas accéder au système.

## Conseils

- Utilisez les filtres pour trouver rapidement les utilisateurs.
- Attribuez les rôles avec attention pour garantir la sécurité.
